output - contains name of the sequence, sequence itself, secondary structure in dot-bracket notation and mfe

2. mfold
Predicts RNA and DNA secondary structure using nearest neighbor thermodynamic rules.

Installation:
http://mfold.rna.albany.edu/?q=mfold/download-mfold

input - file with sequence in fasta-format.

Usage:
mfold SEQ='file_name.fasta'

It creates plenty of different files.
.ct is needed.

Ct2B.pl script should be downloaded:
http://mfold.rna.albany.edu/download/Utils/Ct2B.pl

Usage:
./Ct2B.pl file_name.ct > file_name.b

output - file (file_name.b) with several optimal and suboptimal foldings.
In result file there are sequence itself and a number of secondary structures in dot-bracket notation with their mfe.

How to compare predicted structures with real experimental data:

?. RNAdistance.
Belongs to The ViennaRNA Package.
Calculates distances between RNA secondary structures.

Installation:
http://www.tbi.univie.ac.at/RNA/index.html#download

input - file with structures that need to be compared in dot-bracket notation.

Usage:
RNAdistance < need_to_compare.fasta > result_of_comparison.txt

Example of input file (need_to_compare.fasta):
